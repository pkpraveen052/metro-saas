Goto point of sale settings => To add the transaction charges Tick 'Enable Transaction charge' option.
Select the preferred transaction charge payment method and also select charge type: Percentage or Fixed. Add the specific transaction charge amount.


The selected transaction charge payment method has been added to the payment method.


On the POS screen, you can see the change will apply to the total amount.


Here, you will find the transaction summary, including the credit card amount, the total due amount, and the remaining amount. Click on the validate option.


There are two options available for viewing the receipt: print a receipt or share it via email. As shown in the image, the report is printed in a receipt format where you can see the applied transaction charge.


Select the charge type: Fixed.


Select a product and verify the total cost. The change will be applied to the total amount.


On the receipt, you will find the applied transaction charge.
